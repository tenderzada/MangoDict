#### 如何运行

1. 在setting中修改数据库配置
2. 在线翻译功能可能由于爬虫过期而无法使用（可跳过）

~~~
后台启动：
cd wordbook
python manage.py runserver
前端访问：
http://127.0.0.1:8000/dict/word-list/
~~~
